# TSP Heuristic Algorithms

This repository contains the source code and results for my Bachelor's thesis project on heuristic algorithms for solving the Traveling Salesman Problem (TSP). The project evaluates and compares several heuristic approaches for tackling the NP-hard TSP, focusing on algorithmic efficiency and solution quality.

## Technologies Used

- **Python**: All algorithms and data analysis are implemented in Python.
- **Matplotlib**: Used for generating plots to visualize results and performance.
- **NumPy**: Used for efficient numerical operations.

## How to Run

Clone the repository and run `Main_file.py` with Python 3.x. All dependencies can be installed using pip:

```bash
pip install numpy matplotlib
python Main_file.py
```

## Results

Numerical results and timings for each test case are available in the `results_txt` folder. Visual comparisons are shown in the PNG plots above.
